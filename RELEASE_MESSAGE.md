# WinDesign v1.0.0

## Что нового
1. Перевод всего интерфейса и сообщений в `messages.yml` для полной кастомизации.
2. Стабильное сохранение дизайнов в SQLite (`windesign.db`) с миграцией из старых JSON.
3. Отдельные команды-алиасы: `/prefix`, `/tag`, `/brackets`, `/namecolor`.
4. Корректная работа плейсхолдеров для чата и TAB.
5. Чёткая система пермишенов и доступов к дизайнам.

## Совместимость
1. Сервер: Spigot/Paper 1.16.5.
2. Плагины: PlaceholderAPI, LuckPerms (важно), PlayerPoints (по желанию).

## Установка
1. Скопируйте `WinDesign-1.jar` в `plugins/`.
2. Перезапустите сервер.
3. Настройте `config.yml` и `messages.yml`.

## Важно
1. Плейсхолдеры работают только при наличии PlaceholderAPI.
2. Для покупок за токены нужен PlayerPoints.
